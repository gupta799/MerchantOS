AgentReady

Automated computer-use telemetry for agent-ready commerce

AgentReady runs autonomous computer-use simulations against merchant sites, records every observation/action/state transition, and produces telemetry that helps merchants make their websites MCP-ready and computer-use reliable.

This is not a customer shopping assistant. It is a synthetic CUA lab for merchants.

Architecture Diagram

Merchant OS Dashboard Auto-starts simulation and renders telemetry.
Simulation API POST /api/simulations creates a typed run.
DeepAgents Harness Gemma/OpenAI planner defines the autonomous task.
Computer-Use Provider Scripted, Tzafon, or OpenAI proposes visual actions.
Browser SDK Executes approved actions in the simulated merchant environment.
Telemetry Store Traces, metrics, failures, and MCP recommendations.

Unsafe actions are blocked. Approved actions execute inside the merchant safe root.
Action result + new state
6
TelemetryClassify and summarize.
Backend computes completion, action success, no-op clicks, loops, missing affordances, and MCP readiness.
